# Cadence Coffee

A single-page marketing site for a fictional neighborhood espresso bar — a front-end portfolio piece.

Hand-built with plain HTML, CSS, and vanilla JavaScript. No frameworks, no build step. Open `index.html` in a browser and it runs.

## Highlights

- **Responsive, single-file layout** with a sticky nav and mobile menu
- **Live "Today on bar" brew-spec panel** with a CSS steam animation
- **Tabbed menu**, scroll-reveal animations, and a tasting-note marquee
- **Photo gallery** with a keyboard-accessible lightbox (arrow keys + Esc)
- **Accessibility**: semantic markup, ARIA on interactive widgets, visible focus styles, and `prefers-reduced-motion` support
- **Performance**: lazy-loaded images with explicit aspect ratios (no layout shift)
- **Social-ready**: favicon + Open Graph / Twitter card tags
